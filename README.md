Calculadora_Simples 

Esse código em Python cria um pequeno sistema com login e uma calculadora simples. Primeiro, o programa pede para o usuário digitar um apelido. Ele fica repetindo a pergunta em um loop (while True) até que o usuário digite "Biel". Quando o nome correto é digitado, o programa mostra uma mensagem de boas-vindas e sai desse primeiro loop.

Depois disso, o programa pede uma senha de quatro dígitos. Assim como antes, ele continua pedindo a senha até que o usuário digite 8015. Quando a senha correta é inserida, aparece a mensagem informando que o acesso foi permitido e que o usuário entrou na calculadora.

Na última parte, o programa funciona como uma calculadora. O usuário digita dois números e escolhe a operação matemática (adição, subtração, multiplicação ou divisão). O código calcula o resultado usando condições (if, elif) para verificar qual operação foi escolhida. Também existe uma verificação para evitar divisão por zero, mostrando uma mensagem de erro caso isso aconteça.

Após mostrar o resultado, o programa pergunta se o usuário deseja fazer outra operação. Se a resposta for "s", a calculadora continua funcionando; caso contrário, o programa encerra e mostra uma mensagem de despedida com o nome do usuário.
